# T20 International cricket match prediction Vue.js frontend

## Overview  
- **the application consists of a form that consists of the following fields**
  - Team1
  - Team2
  - Venue
  - Toss winning team
  - gender of players

### the application makes HTTP request to its corresponding Backend API
